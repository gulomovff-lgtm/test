## Ключевые особенности дизайна:

### 1. **Структура**
- Четкие разделители между секциями
- Двойная линия для основных разделов
- Пунктирная линия для подразделов

### 2. **Типографика**
- Заголовки: жирный шрифт, увеличенный размер
- Моноширинный шрифт (Courier New) для таблиц
- Выравнивание по правому краю для сумм

### 3. **Цветовое кодирование**
- 🟢 Зеленый: Полностью оплачено
- 🔴 Красный: Долг
- ⚫ Черный: Обычная информация

### 4. **Информационные блоки**
1. **Заголовок** - название и адрес хостела
2. **Идентификация** - номер и дата чека
3. **Гость** - ФИО, паспорт, страна, комната
4. **Услуги** - детальная разбивка
5. **Оплата** - способы и суммы
6. **Завершение** - благодарность и подпись

### 5. **Адаптивность**
- Ширина 350px (стандарт для чековых принтеров)
- Работает на мобильных устройствах
- Оптимизирован для печати на бумаге 80мм

### 6. **Печать**
- Автоматическая настройка полей
- Скрытие кнопки при печати
- Правильное разбиение на страницы
